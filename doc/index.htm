About This Guide

This guide was developed according to the ISO/IEC 26514:2015 standard "Systems and software engineering. Requirements for designers and developers of user documentation".

The documentation contains all the necessary information for effective use of the Kuroyuki Discord bot.

What is Kuroyuki Bot?

Kuroyuki Bot is a multifunctional Discord bot designed for server management, retrieving user information, and performing administrative tasks.

Key Features

  • Get information about server and users
  • Moderation (timeouts)
  • Connection latency check
  • Slash commands and text commands system
  • Bot status and activity configuration

Important Information

Some commands are available only to the bot owner. Make sure you have appropriate access rights before using administrative commands.

System Requirements

  • Node.js version 18 or higher
  • Discord account
  • Permissions to add bots to server
